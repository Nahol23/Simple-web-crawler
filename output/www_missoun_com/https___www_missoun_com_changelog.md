# Changelog

Find out about Missoun platform latest product changes

#### Aug 31, 2024:

‍ **Missoun Platform**

We've enhanced the backend user experience of the platform, making the configuration process significantly lighter and faster. This improvement ensures a more efficient workflow for all users.

‍

‍

‍

#### July 31, 2024:

‍ **Missoun Platform**

For the eyewear industry, we've introduced facial recognition technology that suggests the ideal eyewear shape for each user. This innovative feature enhances the shopping experience by providing personalized recommendations based on individual facial features.

‍

‍

‍

#### June 30, 2024:

‍ **Missoun Platform**

We've significantly enhanced the 3D rendering capabilities of the Missoun platform. For instance, products like diamonds now display with remarkable clarity and realism, providing a more immersive and detailed visualization.

‍

‍

‍

#### May 31, 2024:

‍ **Missoun Platform**

A new feature allows external suppliers to access customer configurations directly through the Missoun Platform. This functionality streamlines workflows and ensures that products are assembled and manufactured precisely according to customer specifications.

‍

‍

‍

#### Apr 30, 2024:

‍ **Missoun Platform**

We've significantly enhanced the integration of Missoun with Shopify. This update enables seamless synchronization between the Missoun Platform and Shopify, improving the e-commerce experience by streamlining product configuration and purchasing processes.

‍

‍

‍

#### Mar 31, 2024:

‍ **Missoun Platform**

We have enhanced the [WooCommerce integration](https://flebi.configurator.missoun.com/products/b4a7213d-f25f-4cbe-8220-76b420d6e350) functionality of the Missoun platform. In addition to managing dynamic configurations, it is now possible to map SKUs to specific configurations.

‍

‍

‍

#### Feb 29, 2024:

**Missoun Platform**

We have officially released a groundbreaking addition to the Missoun platform: the [Try-On for eyewear,](https://www.missoun.com/resources/elevating-shopping-experiences-with-ar-try-on-transforming-retail-in-the-digital-age) powered by augmented reality (AR). Users can now experience the future of retail firsthand by effortlessly trying on and selecting glasses with unprecedented ease.

‍

‍

#### Jan 31, 2024:

**Missoun Platform**

In this update, we've released several enhancements aimed at elevating the user experience:

1) A new quick link button allows to effortlessly share the configurator with other people,

2) New intuitive zoom and rotation indicators facilitate 3D model visualisation,

3) A new icon clearly indicates when an option is not available,

4) New full screen mode optimized for mobile.

‍

‍

‍

#### Dec 22, 2023:

**Missoun Platform**

In this new release, we have created a set of reports that allow companies to visualize the product configurations saved by their customers. The reports provide a set of filters to easily navigate the data and understand customer preferences.

‍

‍

‍

#### Nov 30, 2023:

**Missoun Platform**

In this new version, we have officially released the functionality to configure a product in different ways and to publish its 3D viewers. With this functionality, it becomes very easy to create a high number of 3D viewers for a digital product catalog.

‍

‍

‍

#### Oct 31, 2023:

**Missoun Platform**

A new wizard helps the user import and validate product features and options from an Excel file. Product features and options can be exported to allow for editing in an external file. This also speeds up the definition of new products that share features with existing products.

‍

‍

‍

#### Sep 30, 2023:

**Missoun Platform**

Product rules definition is now graphical! Rules are defined with a visual configuration screen. The user experience is greatly enhanced, and there is no risk of errors.

‍

‍

‍

#### Aug 31, 2023:

**Missoun Platform**

It is now possible to use web fonts in Missoun. The link where fonts are located can be directly specified in the configuration.

‍

‍

‍

#### Jul 31, 2023:

**Missoun Platform**

Multilanguage is now supported for customer products as well. Product features and options can be displayed in multiple languages!

‍

‍

‍

#### Jun 26, 2023:

**Missoun Platform**

Great improvement to the user experience: as requested by several users, product configuration (features and options) can be now prepared in an Excel file (.csv) and then imported into Missoun 3D CPQ.

‍

‍

‍

#### May 31, 2023:

**Missoun Platform**

You can now define the code format that will be associated to each product configuration. This is extremely useful for manufacturers offering serialized products to their customers.

**Missoun PLM**

It is now possible to select items and categories to export a BOM into a CSV file.

It is now possible to visualise a multi-level BOM as a flat BOM. The user can choose the columns to see in the flat BOM and export data into a CSV file.

‍

‍

‍

#### April 28, 2023:

**Missoun Platform**

In this release we added two crucial features to Missoun 3D CPQ: 1) You can easily configure the pdf document layout that the configurator users will save and download and 2) Product data can be now translated in multiple languages.

‍

‍

‍

#### March 31, 2023:

**Missoun Platform**

Layout configuration: it is now possible to personalize the layout of the configurator, for example by applying specific company colours, fonts, logos.

‍

Native integration with Meta Pixel allows to retrieve deep insights from Meta marketing campaigns.

‍

‍

#### February 28, 2023:

**Missoun Platform**

Localization automatically adapts the 3D configurator to the browser language.

‍

In the configuration environment it is now possible to run new configurations and immediately visualize the effects on the 3D model before publishing.

‍

**Missoun PLM**

Change requests allow to replace multiple items in a BOM at the same time.

‍

#### January 31, 2023:

**Missoun Platform**

Native integration with Microsoft Clarity provides deep insights into users’ behaviour.

‍

**Missoun PLM**

Rollup costs are shown in the item overview page. We also added the possibility to select multiple items to associate a tag directly.

#### December 31, 2022:

**Missoun Platform**

Native integration with HubSpot CRM: contacts who save the configuration are automatically added to HubSpot.

Save a configuration code and reload the configuration in a later stage.

Create and download pdf document of the configuration.

‍

#### November 30, 2022:

**Missoun Platform**

We extended the level of configurability of each product option: it now possible to set multiple actions for a product option and apply textures, materials and colours.

‍

‍

‍

#### October 31, 2022:

**Missoun Platform**

It is now possibile to group products together and show them in a product carousel.

Color circles in the legend can be merged in different groups: color groups are shown in the configurator in separated tabs and they can have a different names.

‍

‍

‍

#### September 30, 2022:

**ERP integration – Missoun PLM** It is now possible to import materials and vendors through API integration with an external system. Integration allows to import and update material costs and material weights as well. As part of the integration process, it is possible to automatically create engineering items from imported materials.

‍

**Item tags – Missoun PLM** Engineering items can be tagged, this functionality allows to group together and filter specific items.

‍

‍

‍

#### August 31, 2022:

**Several improvements - Missoun PLM** For items associated to the same Lifecycle, it is now possibile to choose if the item revision number is automatically incremented (automatic revision numbering). It is now also possibile to define a default list of approvers for the Engineering Change Orders. Several improvements to notification management.

‍

‍

‍

#### July 31, 2022:

**Several improvements - Missoun PLM** When creating an Engineering Change Order it is now possible to add Parent and Children items. In the ECO it is now possible to directly access the details of the engineering items. Several improvements to user and group management and to the item page (e.g. new columns “Created By” and “Created Date”).

‍

‍

‍

#### June 24, 2022:

**Engineering Change Order – Missoun PLM** We released Engineering Change Orders (ECO) in the Missoun platform. Engineering change orders introduce an approval process to authorize changes, they allow to maintain full traceability of changes, with detailed information on the item, what has been changed, who made the change and who has approved it.

‍

‍

‍

#### May 30, 2022:

**Material cost (update from) – Missoun PLM** We added the possibility to keep track of cost variations of supplied materials. With this new feature our customers understand how the cost of their manufactured products has been changing overtime and what is the financial impact of new prices that are applied by their suppliers.

‍

**New user profiles** Improved user management: we introduced specific profiles that can be easily combined to define permissions for each user.

‍

‍

‍

#### April 30, 2022:

**Material import – Missoun PLM** It is now possible to import a list of purchased materials from Excel. The Excel template allows to associate materials to engineering items (Item Number, Item Revision) and to set their properties: Vendor Name, Material Number, Material Name, Material Description, Material Cost, Labour Cost, Overhead Cost, Currency Code, Valid From, Weight, Weight Unit.

‍

‍

‍

#### March 31, 2022:

**Plant model – Missoun PLM** We added the ability to define the model of your plants, by following the ISA-95 structure: enterprise, site, area, work centers, work units.

‍

‍

‍

#### February 28, 2022:

**Missoun Platform**

**‍** Do you want to add new colors to the product you are showing in the 3D configurator? With this new release, it is much easier than before. There is no need to modify the 3D model itself; you can add new colors directly within the 3D configurator.

‍

‍

‍

#### January 31, 2022:

**Missoun Platform**

**‍** In addition to selecting available options, Missoun 3D CPQ now supports numerical features for configuring the product. For example, bicycle manufacturers can add the feature “cyclist height” to their configurator so that the user can insert their height in cm in the data form.

‍

It is now possible to create and personalize emails sent from the Missoun platform to your customers.

‍

‍

‍

#### December 31, 2021:

**Missoun Platform** We added a new product to our portfolio: Missoun 3D CPQ, the 3D product configurator for complex manufacturing. With Missoun 3D CPQ, customers enjoy product configuration with 3D visualization, real-time price calculation, and immediate quote creation.

‍

‍

‍

#### November 30, 2021:

**Production process – Missoun PLM** In this release we added the ability to define the production process for your products: starting from the EBOM it is possible to define process plans, operations, consumed materials and the manufacturing [BOM](https://www.missoun.com/150-bom).

‍

‍

‍

#### October 31, 2021:

**Cost and vendor management – Missoun PLM** You can now monitor the real cost of your products by taking into account the cost of the purchased materials, the labour costs and the indirect costs.

For the supplied materials it is also possible to manage different vendors with their related costs.

‍

**Where Used – Missoun PLM** In the details of the item you will find a new tab “Where Used” that shows which BOM is using that version of the item.

‍

‍

‍

#### September 30, 2021:

**Variant management – Missoun PLM** With Missoun PLM you can now offer configuration options to customers. Missoun PLM removes the complexity of managing product variants manually: you can configure options for your product to define the available variations. When the user selects the desired product options, Missoun PLM resolves the [bill of material](https://www.missoun.com/bill-of-material) for production.

‍

‍

‍

#### August 31, 2021:

**Missoun Connect** Your customers can open service requests, share their experience with the product, establish a dialogue with your team. Your engineering and production teams can now access customer feedbacks and use such inputs to improve products and processes.

‍

‍

‍

#### July 31, 2021:

**Item Redlining – Missoun PLM**

Changes applied to an item are highlighted when creating a new item revision: new values are shown in green, old values are shown in red with strikethrough.

‍

**Audit trail of project tasks – Missoun PLM**

When editing a project task, changes are logged and visible in the “Activity” tab. The audit trail functionality keeps track of the timestamp, the user and what was changed.

‍

**Link project work item to an item – Missoun PLM** It is now possible to link a work item to an engineering item. When viewing the details of an engineering item in Missoun Design, the associated work items are shown on the right side of the screen in the “Work Items” tab.

‍

‍

‍

#### Q1 and Q2, 2021:

**Project management – Missoun PLM**

Users can improve productivity and collaboration by managing projects with Missoun PLM: it is possible to create collaborative projects, add members and guest users (e.g. external suppliers), assign project tasks and monitor their execution. In addition, users can improve collaboration with others by adding comments and files to tasks.

‍

**Engineering processes – Missoun PLM**

Missoun PLM allows managing items and BOMs by establishing a real collaboration within your teams. It is possible to create items and BOMs manually and also to import them from Excel through a simple guided process. Files can be uploaded and attached to items to ensure that the team can easily find whatever relates to your product. Consistency of item changes is managed through the item revision process and the transition between 3 statuses: draft, released and obsolete. Items can be organized in different categories to facilitate their utilization: it is possible to assign specific lifecycles and units of measures to item categories.
